How to Get Duration of Video

April 09, 2022 • 2 min read • #FFmpeg, #video, #duration

一.下载FFmpeg

去FFmpeg官网 https://ffmpeg.org/download.html 下载

二.添加到环境变量

将下载的ffmpeg和ffprobe都复制到/usr/local/bin文件夹下

cd ~/Downloads/
# 查看下载的文件
ls -a | grep "ffmprob"
ls -a | grep "ffmpeg"
# 复制ffmpeg
sudo cp ./ffmpeg /usr/local/bin
sudo chmod ugo+x /usr/local/bin/ffmpeg
# 复制ffprobe
sudo cp ./ffprobe /usr/local/bin
sudo chmod ugo+x /usr/local/bin/ffprobe

到这一步已经可以在任何路径下使用ffmpeg命令了

如果没有可能是你没有将/usr/local/bin添加到 $PATH

sudo vi ~/.bash_profile

在文件末尾添加以下内容即刻:

export PATH="/usr/local/bin:$PATH"

三.查看视频时长

ffprobe -v error -show_entries format=duration -of default=noprint_wrappers=1:nokey=1 -i main-video.mp4
